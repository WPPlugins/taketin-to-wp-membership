TAKETIN Marketing Platformと連携したメンバーサイトをWordpressで作成するプラグインです。

== Description ==

<div>
  TAKETIN Marketing Platformと連携したメンバーサイトをWordpressで作成する事ができます。
</div>

<div>
  TAKETIN Marketing Platformで作成した商品毎に会員サイトを用意することができます。
</div>

== Installation ==

1. Upload \`taketin-to-wp-membership\` to the \`/wp-content/plugins/\` directory

2. Activate the plugin through the 'Plugins' menu in WordPress
== Frequently Asked Questions ==

**無料で使えますか？**

はい、無料で使えますが、TAKETINシステムを導入の方のみ機能させることが出来ると思います。
